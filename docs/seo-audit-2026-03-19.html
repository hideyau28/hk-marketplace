🚨 索引緊急問題

Google 完全冇索引 wowlix.com。呢個係最 critical 嘅問題——再靚嘅 SEO 設定都冇用如果 Google 睇唔到你個站。

可能原因（按可能性排序）：

1. 未設定 Google Search Console — 未驗證域名擁有權，Google 唔會主動爬

2. DNS / 域名未正確指向 Vercel — wowlix.com 可能仲未 live

3. Sitemap 未提交 — 雖然 robots.txt 有聲明，但手動提交可以加速索引

4. force-dynamic 配合 Vercel — 如果頁面 response time 太慢，Googlebot 可能放棄爬
